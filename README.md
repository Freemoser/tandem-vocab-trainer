# Thai Learning Memory

A small browser-based Thai learning card game for solo practice and tandem lessons.

## Run locally

```bash
python3 -m http.server
```

Then open:

```text
http://127.0.0.1:8000/
```

## Project structure

- `index.html` - app shell
- `style.css` - responsive card and game styling
- `app.js` - game logic
- `vocabulary.json` - local vocabulary database

## Notes

This is a static app. It has no backend, no login, no database server, and no stored secrets.
